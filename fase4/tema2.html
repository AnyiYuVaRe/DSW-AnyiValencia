Frameworks CSS


Podemos definir un framework para aplicaciones web es un framework que sirve para el desarrollo web: aplicaciones web, sitios web dinámicos y servicios web.
Los frameworks proporcionan herramientas, bibliotecas, plantillas, códigos y aplicaciones de ejemplos, etc., que facilitan el desarrollo web.
El Framework CSS está relacionado con:

  • Aplicación web
  • Desarrollo web
