Fundamentos CSS



Este lenguaje sirve para definir la estructura de la págia. El CSS nos sirve para:

  • Sirve para dar estilo a los elementos

  • Funciona a base de selectores y propiedades

  • Le da color, tamaño y espacios a los elementos

  • Aplicar estilos

  • Selector

  • Qué quieres cambiar: propiedad.

  • Valor de propiedad h1, h2, h3

Para poder realizar todo lo que antes mensionamos, hay que utilizar algunas de las siguientes propiedades CSS:

  • font-style

  • font-weight: 400;

  • (500, 700)

  • font-family: Open Sans;

  • font-size: 14px;

  • text-align: center;

  • text-transform: uppercase
